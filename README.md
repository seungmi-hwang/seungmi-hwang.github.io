# Personal Portfolio Website

## Purpose
This website is a personal portfolio created to introduce myself, showcase my skills and projects, and present interactive visualizations.  
It serves as a portfolio for my work in visual effects, and visualization design.

The site includes:
- A main portfolio page with personal information, skills, projects, and contact details
- A separate visualization page that focuses only on SVG-based visualizations created with JavaScript

---

## Technologies Used
This website is built using the following technologies:

- HTML5  
  Used to structure the content of the website, including sections such as About, Skills, Projects, and Contact.

- CSS3 
  Used for styling and layout. The site uses a responsive layout to ensure it works well on different screen sizes.

- JavaScript 
  Used to add interactivity and generate SVG-based visualizations.
  - Smooth scrolling and navigation highlighting on the main page
  - Dynamic SVG visualizations and creative SVG art on the visualization page

- SVG
  Used for data visualization and creative visual elements generated through JavaScript.

- GitHub Pages
  Used to host and deploy the website online.


## Site Structure & Navigation

### Main Page (`index.html`)
- Navigation bar: at the top allows users to jump to different sections:
  - About
  - Skills
  - Projects
  - Contact
- A hero section: introduces the site and highlights key interests.
- Content is organized into clearly separated sections for easy reading.

### Visualization Page (`visualizations.html`)
- This page is dedicated only to visualizations.
- It includes:
  - One SVG data visualization created entirely with JavaScript
  - One creative SVG artwork generated and animated using JavaScript
- No resume or textual content is included on this page, following the project requirements.

### JavaScript Files
- `main.js`: Handles basic interactivity on the main page (smooth scrolling and active navigation highlighting).
- `vis.js`: Generates SVG drawings and visualizations using JavaScript.
